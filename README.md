## ğŸ’¡ Rest API HTTP Method

* Retrieve all Users
  - REST API : /users
  - HTTP Method :GET

* Create a User
  - REST API : /users
  - HTTP Method :POST

* Retrieve one User
  - REST API : /users/{id}
  - HTTP Method :GET

* Delete a Users
  - REST API : /users/{id}
  - HTTP Method :DELETE

* Retrieve all posts for a User
  - REST API : /users/{id}/posts 
  - HTTP Method :GET

* Create a posts for a User
  - REST API : /users/{id}/posts
  - HTTP Method :POST

* Retrieve details of a User
  - REST API : /users/{id}/posts/{post_id}
  - HTTP Method :GET
